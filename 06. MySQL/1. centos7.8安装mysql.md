# centos7.8安装mysql

## 离线安装

### 一、环境准备与检查

#### 1.1 系统检查
```bash
# 查看系统版本
cat /etc/redhat-release

# 检查是否已安装MySQL/MariaDB
rpm -qa | grep -i mysql
rpm -qa | grep -i mariadb

# 如果存在旧版本，彻底卸载
sudo yum remove -y mysql* mariadb*
```

#### 1.2 安装依赖包
```bash
# 安装基本依赖
sudo yum install -y libaio net-tools perl numactl

# 如果有离线依赖包，使用rpm安装
sudo rpm -ivh /path/to/your/libaio*.rpm
```

### 二、MySQL安装步骤

### 2.1 获取MySQL安装包
从官网https://downloads.mysql.com/archives/community/

下载 `mysql-5.7.19-1.el7.x86_64.rpm-bundle.tar` 并上传到服务器。

### 2.2 解压安装包
```bash
# 创建安装目录并解压
sudo mkdir -p /opt/mysql-install
sudo tar -xvf mysql-5.7.19-1.el7.x86_64.rpm-bundle.tar -C /opt/mysql-install
cd /opt/mysql-install
```

### 2.3 安装RPM包
```bash
# 按顺序安装必要的RPM包
sudo yum localinstall -y \
    mysql-community-common-5.7.19-1.el7.x86_64.rpm \
    mysql-community-libs-5.7.19-1.el7.x86_64.rpm \
    mysql-community-libs-compat-5.7.19-1.el7.x86_64.rpm \
    mysql-community-client-5.7.19-1.el7.x86_64.rpm \
    mysql-community-server-5.7.19-1.el7.x86_64.rpm
```

### 2.4 创建MySQL用户和数据目录
```bash
# 创建mysql用户（如果不存在）
sudo groupadd mysql
sudo useradd mysql -g mysql

# 创建数据目录（建议使用独立分区）
sudo mkdir -p /home/mysql/data
sudo chown -R mysql:mysql /home/mysql
sudo chmod -R 755 /home/mysql
```

## 三、MySQL配置

### 3.1 配置文件设置
编辑 `/etc/my.cnf`：

```ini
[mysqld]
# 基本设置
user = mysql
port = 33061
basedir = /usr
datadir = /home/mysql/data
socket = /var/lib/mysql/mysql.sock
pid-file = /var/run/mysqld/mysqld.pid

# 日志设置
log-error = /home/mysql/mysql_error.log
slow_query_log = 1
slow_query_log_file = /home/mysql/mysql_slow.log
long_query_time = 2

# 字符集设置
character-set-server = utf8mb4
collation-server = utf8mb4_general_ci

# InnoDB设置
innodb_buffer_pool_size = 1G
innodb_log_file_size = 256M

# 网络设置
bind-address = 0.0.0.0
max_connections = 500

[client]
default-character-set = utf8mb4
socket = /var/lib/mysql/mysql.sock

[mysql]
default-character-set = utf8mb4
```

### 3.2 创建必要的目录并设置权限
```bash
# 创建socket和pid文件目录
sudo mkdir -p /var/lib/mysql
sudo mkdir -p /var/run/mysqld

# 设置权限
sudo chown -R mysql:mysql /var/lib/mysql
sudo chown -R mysql:mysql /var/run/mysqld
```

## 四、初始化MySQL

### 4.1 初始化数据目录
```bash
# 初始化MySQL（重要：记录生成的临时密码）
sudo mysqld --initialize --user=mysql --datadir=/home/mysql/data

# 查看临时密码
sudo grep 'temporary password' /home/mysql/mysql_error.log
```

### 4.2 启动MySQL服务
```bash
# 启动服务并设置开机自启
sudo systemctl start mysqld
sudo systemctl enable mysqld

# 检查服务状态
sudo systemctl status mysqld
```

## 五、安全设置与远程访问

### 5.1 修改root密码
```bash
# 使用临时密码登录
mysql -u root -p

# 修改root密码
ALTER USER 'root'@'localhost' IDENTIFIED BY 'YourStrongPassword123!';
FLUSH PRIVILEGES;
```

### 5.2 配置远程访问
```sql
-- 创建远程访问用户（推荐使用专用用户而非root）
CREATE USER 'admin'@'%' IDENTIFIED BY 'YourStrongPassword123!';
GRANT ALL PRIVILEGES ON *.* TO 'admin'@'%' WITH GRANT OPTION;
FLUSH PRIVILEGES;

-- 或者允许root远程访问（不推荐用于生产环境）
-- CREATE USER 'root'@'%' IDENTIFIED BY 'YourStrongPassword123!';
-- GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;
-- FLUSH PRIVILEGES;
```

### 5.3 防火墙配置
```bash
# 开放MySQL端口
sudo firewall-cmd --permanent --add-port=33061/tcp
sudo firewall-cmd --reload

# 查看防火墙规则
sudo firewall-cmd --list-all
```

## 六、验证安装

### 6.1 本地连接测试
```bash
mysql -u root -p -P 33061
```

### 6.2 远程连接测试
```bash
# 从其他机器测试连接
mysql -u admin -h 10.20.140.61 -P 33061 -p
```

### 6.3 检查MySQL状态
```sql
-- 查看版本信息
SELECT VERSION();

-- 查看用户权限
SELECT user, host FROM mysql.user;

-- 查看运行参数
SHOW VARIABLES LIKE 'port';
SHOW VARIABLES LIKE 'datadir';
```

## 七、常见问题解决

### 7.1 端口冲突问题
```bash
# 检查是否有其他MySQL实例运行
ps -ef | grep mysqld | grep -v grep

# 如果发现多个实例，停止不需要的实例
sudo kill <不需要的进程ID>

# 确认只有一个MySQL实例监听端口
ss -tnlp | grep mysqld
```

### 7.2 权限问题
```bash
# 确保所有相关目录权限正确
sudo chown -R mysql:mysql /home/mysql
sudo chown -R mysql:mysql /var/lib/mysql
sudo chown -R mysql:mysql /var/run/mysqld
```

### 7.3 SELinux问题
```bash
# 检查SELinux状态
sudo getenforce

# 如果是Enforcing模式，调整策略或临时禁用
sudo setenforce 0  # 临时禁用
sudo semanage port -a -t mysqld_port_t -p tcp 33061  # 添加端口策略
```

## 八、后续维护建议

```

### 8.2 日志监控

sudo tail -f /home/mysql/mysql_error.log

# 监控慢查询日志

### 8.3 定期维护

```

## 总结

通过以上步骤，您应该能够成功在CentOS 7.6上安装和配置MySQL 5.7。关键点包括：

1. **彻底卸载旧版本**，避免冲突
2. **使用RPM Bundle包**进行离线安装
3. **正确配置数据目录和权限**
4. **合理设置MySQL配置文件**
5. **妥善处理防火墙和SELinux**
6. **创建专用远程访问用户**而非直接使用root

这份指南包含了我们沟通中遇到的所有问题及其解决方案，应该能够帮助您顺利完成MySQL的安装和配置。

   - `yum` 是一个在许多Linux发行版中用于包管理的工具，它可以帮助你安装、升级和管理软件包。

### 2. MySQL配置

#### 2.1 启动mysql
执行命令
```bash
systemctl start mysqld
#或
systemctl start mysqld.service
```
查看mysql运行状态

```bash
systemctl status mysqld
```

#### 2.2 设置密码

首先查看mysql的初始密码。

```bash
grep "password" /var/log/mysql/mysqld.log
```

如果出现以下信息：

```bash
[Warning] [MY-010453] [Server] root@localhost is created with an empty password ! Please consider switching off the --initialize-insecure option.
```

说明mysql root用户没有设置初始密码。需要设置密码。

```bash
alter user 'root'@'localhost' identified by 'new_passwd';
```
