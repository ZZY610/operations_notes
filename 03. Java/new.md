# new
李牧是第一个被抓的，当时李牧接到赵王密旨要去邯郸开会，讨论抗秦的事情。他刚走近赵王宫时，下属司马尚即被留在门外。
李牧感到事情有些不大对头，但也没在意。当他快走近大门时，专门对付他的王室禁卫军立即走了过来。
几名侍卫在走廊里把他扭住，李牧惊慌失措，一边大声说“我是来开会的，你们要干什么？”一边拳打脚踢，拼命进行反抗。禁军个个身手不凡，李牧很快就被他们制服，被扭着双臂押到了王宫东侧。
在这里，等待他的赵葱把诏书念了一遍。还没等赵葱念完，李牧突然大吼一声，挣脱禁军们的扭缚，向五六步远地方的郭开猛扑过去。李牧在雁门带了十几年的兵，练过武，一旦扑过去，打伤了郭开这还了得？郭开久经沙场不慌不忙的冷眼看着李牧的疯狂举动。
在这千钧一发之际，一旁的颜聚反应迅速，猛冲上去把李牧扑倒，把他死死摁住。在抓捕“代地边军派”的成员过程中，对李牧的惩罚是最严格的。
