# minecraft 1.20.1服务器搭建
## 1、系统基本信息

- 操作系统：Ubuntu 22.04
- CPU：4核8gb
- 存储空间：100gb
- MC版本：1.20.1

## 2、部署流程

服务器配置（**Ubuntu 22.04 | 4核8GB内存 | 100GB存储**）


### 2.1 服务器环境准备
#### 1. 连上服务器  
```bash
ssh ubuntu@你的公网IP
```

#### 2. 装 Java 17  
```bash
# Ubuntu / Debian
sudo apt update
sudo apt install -y openjdk-17-jre-headless

# CentOS / Rocky
# yum install -y java-17-openjdk-headless
```


##### 3.1 关于run.sh脚本

```sh
ubuntu@VM-16-12-ubuntu:~/mc$ cat run.sh 
#!/usr/bin/env sh
# Forge requires a configured set of both JVM and program arguments.
# Add custom JVM arguments to the user_jvm_args.txt
# Add custom program arguments {such as nogui} to this file in the next line before the "$@" or
#  pass them to this script directly
java @user_jvm_args.txt @libraries/net/minecraftforge/forge/1.20.1-47.2.0/unix_args.txt "$@"
```


##### 3.2 `#!/usr/bin/env sh`
- **作用**：Shebang（解释器指令），告诉系统用哪个程序来执行这个脚本
- **为什么用 `env sh`**：
  - `env` 会在系统的 PATH 中查找 `sh`
  - 比直接写 `/bin/sh` 更兼容不同 Linux 发行版
  - 确保使用用户的默认 shell（通常是 bash 或 dash）

##### 3.3 注释部分
```sh
# Forge requires a configured set of both JVM and program arguments.
# Add custom JVM arguments to the user_jvm_args.txt
# Add custom program arguments {such as nogui} to this file in the next line before the "$@" or
#  pass them to this script directly
```
- **说明**：
  - 第一行：Forge 需要同时配置 JVM 参数和程序参数
  - 第二行：自定义 JVM 参数（如内存设置）请写到 `user_jvm_args.txt`
  - 第三行：自定义程序参数（如 `nogui`）可以：
    - 直接加在 `./run.sh nogui` 后面
    - 或者修改这个脚本在 `"$@"` 前添加

##### 3.4 核心启动命令
```sh
java @user_jvm_args.txt @libraries/net/minecraftforge/forge/1.20.1-47.2.0/unix_args.txt "$@"
```

###### ① java
- 启动 Java 虚拟机（需要 Java 17+）

###### ② `@user_jvm_args.txt`
- **@符号**：Java 9+ 的特性，表示从文件读取参数
- 读取你自定义的 JVM 参数，比如：
  ```
  -Xms2G -Xmx4G -XX:+UseG1GC
  ```

###### ③ @libraries/net/minecraftforge/forge/1.20.1-47.2.0/unix_args.txt
- **Forge 预配置的参数文件**，包含：
  - **类路径**（-cp）：所有需要的 jar 文件
  - **主类**：`net.minecraftforge.server.ServerMain`
  - **其他 JVM 参数**：如 `-Dforge.enableGameTest=true`

###### ④ `"$@"`
- **传递脚本参数**：允许你运行时添加额外参数
- **示例**：
  - `./run.sh nogui` → 禁用 GUI
  - `./run.sh --help` → 显示帮助

这个脚本是一个**智能启动器**，它：
1. 读取你的自定义 JVM 设置
2. 加载 Forge 预配置的所有必要参数
3. 允许你灵活添加启动参数
4. 跨平台兼容（Linux/macOS）
